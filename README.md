# Portfólio Web Pessoal

Este é um projeto de portfólio web desenvolvido como parte de uma atividade acadêmica. O objetivo foi criar uma página estática utilizando **HTML** e **CSS**, representando um portfólio pessoal com informações como biografia, habilidades, projetos e formas de contato.

## Tecnologias Utilizadas

- HTML5
- CSS3

> Nenhum framework, biblioteca ou linguagem adicional foi utilizada.

## Funcionalidades

- Página inicial com uma breve apresentação
- Seção de **sobre mim**
- Listagem de **habilidades**
- Galeria de **projetos**
- Sessão de **contato** (sem funcionalidade dinâmica)

## Objetivo do Projeto

Desenvolver uma página web estática como introdução ao desenvolvimento front-end, praticando a estruturação de conteúdo com HTML e a estilização com CSS.

##  Como visualizar

1. Clone o repositório ou baixe os arquivos:
   git clone https://github.com/yuriribeirodev/projeto-portfolio.git
2.  Abra o arquivo 'index.html' em qualquer navegador.
